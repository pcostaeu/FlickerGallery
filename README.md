## FlickerGallery

Android MVVM example with Clean Architecture

Libraries used:
* Koin (Service locator)
* RxJava2 (Reactive streams)
* Retrofit (API calls and caching requests)
* Moshi (JSON parsing)
* Glide (Image loading and image caching)
